Examples
========

This folder contains notebooks that illustrate how certain common data types are represented and stored in the USID format.

* `1D - Spectrum or curve <./spectrum.ipynb>`_
* `2D - Image <./image.ipynb>`_
* `3D - Hyperspectral image <./spectral_image.ipynb>`_
* `3D - Image stack <./image_stack.ipynb>`_
* `3D - Video <./video.ipynb>`_
